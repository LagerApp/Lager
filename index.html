What is Lager?

Lager is a logging service that provides a mobile web interface for your devices.

It runs on your server and sets up a authenticable web interface where you can add servers and service logs to monitor.

How it works

  1. Clone LagerApp onto the server you want Sinatra to run on.

  2. Run: 

    bundle exec rake db:setup
    to set up the Sinatra dependencies and instance.

  3. Another script will be generated to run on the servers that you want to monitor.

  4. Unix users with the correct permissions and access control with public/private keys will be setup.

  5. Enjoy reading logs from your mobile device. üçª
