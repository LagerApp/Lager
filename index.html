What is Lager?

Lager is a logging service that provides a mobile web interface for your devices.

It runs on your server and sets up a authenticable web interface where you can add servers and service logs to monitor.

Dependencies

  • Ruby 2.2

How it works

  1. Clone LagerApp onto the server you want Sinatra to run on.

  2. Run bundle install to install the gem dependencies. Run `gem install bundler` if you do not have Bundler installed.

  3. Run setup.sh. This will set up the sqlite database, a default username and password, and another script to run on the servers you want to monitor. The generated script makes use of a private/public key pair and a restricted unix account that only has access to log folders on other servers.

  4. Run ruby app.rb to start the server. We recommend using Supervisor to run the server in the background.
